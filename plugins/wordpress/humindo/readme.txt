== Description ==

<b>KOSTENLOSER LIVECHAT</b>

Nutzen Sie den kostenlosen Livechat und erweitern Sie als Berater Ihre Reichweite. Sparen Sie Zeit und halten Sie Erstgespräche direkt online per Video oder Chat ab. Integrieren Sie die Chatlösung von humindo direkt auf Ihrer eigenen Homepage und stehen Sie gleichzeitig auf dem Beraternetzwerk von humindo zur Verfügung.

<b>AUS GLÜCKLICHEN BESUCHERN WERDEN ZUFRIEDENE KUNDEN</b>

Sie und Ihr Team können zu jeder beliebigen Zeit im Chat zur Verfügung stehen. Sie müssen dazu nur Ihre eigene Webseite öffnen und den Chat starten. Nutzen Sie so die Zeiten, in denen Sie ohnehin auf Ihrem Computer arbeiten. Sie erhalten Benachrichtigungen bei eingehenden Nachrichten. Ist keiner Ihrer Berater online, dann können Besucher den Chat nutzen um Sie per Email zu kontaktieren. So erhalten Sie die wichtigen Kontaktdaten und verpassen keine Chance. Machen Sie es neuen Besuchern besonders leicht Sie zu kontaktieren.

<b>PROBLEME LÖSEN, FRAGEN BEANTWORTEN, AUFTRÄGE GENERIEREN</b>

Kunden schätzen eine schnelle Antwortzeit. Generieren Sie neue Aufträge genau dann, wenn Kunden sich über Ihre Dienstleistung informieren und eine Lösung für ihr Problem suchen. Besucher Ihrer Webseite können Sie sofort kontaktieren und müssen keine langen Antwortzeiten in Kauf nehmen. Räumen Sie Bedenken aus und bauen Sie direkt eine Beziehung auf. Konvertieren Sie nachhaltig interessierte Besucher zu neuen Kunden.

<b>LIVE-BERATUNG</b>

Halten Sie Erstgespräche online ab und ermöglichen Sie Ihren Bestandskunden eine schnelle Kontaktaufnahme. Starten Sie eine Videokonferenz direkt von Ihrer eigenen Webseite aus. Sparen Sie sich und Ihren Kunden die mühsame Installation und Einrichtung von externen Programmen. Steigern Sie die Zufriedenheit Ihrer Kunden indem Sie ihnen bei Problemen schnell zur Seite stehen.

<b>DER UNTERSCHIED</b>

Mit der Chat-Integration von humindo haben Sie die Möglichkeit auf Ihrer eigenen Webseite sowie auf dem Beraternetzwerk von humindo gleichzeitig zur Verfügung zu stehen. Besucher Ihrer Webseite als auch von humindo.at haben die Möglichkeit Sie zu kontaktieren. Mit Ihrer Präsenz erreichen Sie mehr Kunden und kommen zu mehr Aufträgen. Das macht humindo zum größten Beraternetzwerk - seien Sie jetzt dabei.
Entscheiden Sie zu jeder Zeit selbst, ob Sie nur auf Ihrer Webseite oder auch auf humindo sichtbar sein wollen.

== Installation ==
Die Installation des humindo Livechats ist einfach, befolgen Sie die folgenden Anweisungen:

1. Laden Sie die Plugin Dateien in Ihr Verzeichnis `/wp-content/plugins/humindo`
2. Aktivieren Sie das Plugin im 'Plugins' Menü in WordPress
3. Erstellen Sie Ihr Profil auf https://www.humindo.at
4. Nun erscheint der Livechat Button im rechten unteren Bereich Ihrer Webseite.

== FAQ ==

= Wird meine Wordpress Seite langsamer? =
Das humindo Widget lädt asynchron (wie Google Analytics) - Es beeinflusst die Performance und Ladezeiten Ihrer Webseite somit nicht.

= Welche Browser werden unterstützt? =
Es werden alle gängigen Browser unterstützt.
